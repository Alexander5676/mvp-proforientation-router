# MVP профориентации — React Router + Tailwind (Vite)

Приложение разбито на страницы с маршрутизацией через React Router DOM.

## Как запустить
```
npm install
npm run dev
```

## Страницы
- `/` — Онбординг
- `/test` — Тест интересов
- `/results` — Результаты
- `/card/:id` — Карточка профессии
- `/plan` — Персональный план

Маршруты можно легко адаптировать под SPA или Next.js.
